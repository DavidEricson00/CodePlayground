# Code Playground

This repository contains small and simple projects built for learning, experimentation, and fun.  
The focus here is not on complexity, but on practicing fundamentals, exploring ideas, and improving hands-on skills over time.

## Projects

### MyMe

A simple profile generator built with pure JavaScript and DOM manipulation.

**What it does:**
- Collects user data through a form
- Displays the information dynamically on the page
- Updates the UI based on user input

**Concepts explored:**
- HTML forms (input, select, checkbox, radio, textarea)
- DOM manipulation with vanilla JavaScript
- Event handling (form submission)
- Dynamic content rendering

---

### WebData

A basic interactive guide for HTML tags and web concepts.

**What it does:**
- Organizes HTML tags into categories
- Provides navigation to detailed pages for each tag
- Includes UI features like search, dark mode, and reading mode

**Concepts explored:**
- HTML structure and semantic organization
- Navigation between pages
- Basic UI interactivity with JavaScript
- Styling and layout with CSS
- Accessibility-focused features (reading mode, contrast)

## Notes

The code is intentionally kept simple and straightforward.  
More advanced and production-ready projects are available in other repositories.
