# WebData

Catálogo/manual de referência das tags estudadas em aula.

O site deve mostrar as tags funcionando na própria página.

---

## 📌 Objetivo

Criar um site com páginas ou seções que expliquem as seguintes tags HTML:

- html  
- head  
- body  
- p  
- h1–h6  
- ul  
- ol  
- li  
- a  
- img  
- span  
- strong  
- em  
- hr  

---

## 📄 Requisitos por Tag

Cada tag deve conter:

- Nome da tag (ex: `<p>`)  
- Um parágrafo explicando sua função  
- Um exemplo funcionando (visível na página)  
- Uma imagem ou ícone representativo  

---

## 🧭 Página Principal

- Deve conter uma lista (`ul`) com links para as páginas ou seções das tags  
- Funciona como um sumário do site  

---

## 🗂️ Organização das Tags

### Estrutura
- html  
- head  
- body  

### Conteúdo
- p  
- h1–h6  
- strong  
- em  
- span  
- a  
- hr  

### Listas
- ul  
- ol  
- li  

### Mídia
- img  

---

## 🎨 CSS

- Ajustar margens e paddings para melhor espaçamento  
- Personalizar listas (`list-style-type`)  
- Remover sublinhado dos links  
- Alterar cor dos links ao passar o mouse  

### ⭐ Extra (bônus)
- Implementar modo claro e escuro  

---

## ⚙️ JavaScript

### 🔍 Busca por texto
- Campo de pesquisa que destaca a tag digitada na lista  

### 📖 Modo de leitura
- Botão para alternar tamanho do texto (16px ↔ 20px)  

### 🖼️ Troca de imagem
- Ao passar o mouse sobre uma imagem, trocar por outra relacionada  

---

## 📌 Observações

- Usar `<h1>` como título principal  
- Usar `<hr>` para separar seções  
- O site deve ser organizado, legível e funcional
